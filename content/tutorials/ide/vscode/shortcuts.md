title: Keyboard shortcuts
---

## Navigation

- Go to settings: `Ctrl + ,`
- Go to explorer: `Ctrl + Shift + E`
- Go to version control: `Ctrl + Shift + G`
- Go to debugger: `Ctrl + Shift + D`
- Go to folder find and replace: `Ctrl + Shift + F`
- Go to extensions: `Ctrl + Shift + X`


## Find and replace

- In the whole folder
  - Find: `Ctrl + Shift + F`
  - Replace: click on `>` of the search box to show the replace box
- In the current file
  - Find: `Ctrl + F`
  - Replace: click on `>` of the search box to show the replace box

## Command palette

- Go to command palette: `Ctrl + Shift + P`

## Working with multiple files

- Switch between tabs
  - Forward: `Ctrl + Tab`
  - Backward: `Ctrl + Shift + Tab`
- Search file by name: `Ctrl + P`

## Working with one file

- Go to line in the current file: `Ctrl + G`
- Close the file: `Ctrl + W`

## Terminal

- Toggle terminal: `Ctrl + ~`

## Other

- Copy a line: put the cursor at the line and press `Ctrl + C`
- Select a word: put the cursor at the word and press `Ctrl + D`
