---
title: Keyboard shortcuts
---

This post summarizes some of the most common keyboard shortcuts that can significantly boost your productivity while working with VS Code. In case you forget or want to search for a particular hotkey, you can press `Ctrl + K Ctrl + S`. It will show the key binding table together with a search box for you to type in the keywords.

Note that the hotkeys introduced in this tutorial is intended for Windows and Linux users. If you are using macOS, they are pretty similar with `Ctrl` being replaced by `Cmd` and `Alt` being replaced by `Option`. However, there might be minor differences, in which case you should Google to find the correct key binding.

## Navigation

- Go to Settings: `Ctrl + ,`
- Go to Explorer: `Ctrl + Shift + E`
- Go to Version Control: `Ctrl + Shift + G`
- Go to Debugger: `Ctrl + Shift + D`
- Go to Find and Replace (global): `Ctrl + Shift + F`
- Go to Extensions: `Ctrl + Shift + X`

## Search

- Search for a file by name: `Ctrl + P`
- Search for text globally: `Ctrl + Shift + F`
- Search for text in the current file: `Ctrl + F`
- Go to line number: `Ctrl + G`
- Search for symbols: `Ctrl + Shift + O`

## Command palette

- Go to command palette: `Ctrl + Shift + P`

## Creation

- New VS Code instance: `Ctrl + Shift + N`
- New unnamed file: `Ctrl + N`
- New file at specific subfolder: `Ctrl + Alt + N` (need Advanced New File extension)

## Terminals

- Toggle terminal: `Ctrl + ~`

## Editor

### Operations on tabs or groups

- Switch between tabs: `Ctrl + Tab` (forward) or `Ctrl + Shift + Tab` (backward)
- Close the current tab: `Ctrl + W`
- Close all tabs in the current group: `Ctrl + K W`
- Close all tabs: `Ctrl K Ctrl W`
- Go to the leftmost editor group: `Ctrl + 1`
- Go to center editor group: `Ctrl + 2`
- Go to the rightmost editor group: `Ctrl + 3`

### Operations on texts

- Copy the current line: `Ctrl + C`
- Select the current word: `Ctrl + D`
- Move current line: `Alt + Up/Down`
- Duplicate current line: `Shift + Alt Up/Down`
- Expand selection: `Shift + Alt + Right`
- Shrink selection: `Shift + Alt + Left`
- Toggle word wrap: `Alt + Z`
- Toggle comments: `Ctrl + /`

## Modes

- Toggle zen mode: `Ctrl + K Z` (or double `Esc` to exit)
