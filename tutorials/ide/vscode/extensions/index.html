To install a new extensions, open Extensions by pressing Ctrl + Shift + X, then typing the extension name to search and install. The following are some of my favorite extension that boosts productivity.

Prettier

This extension (by Prettier) is a code formatter. One of its cool feature is Format on save allowing you to format your code whenever you hit Ctrl + S to save your file.

Advanced new file

This extension (by patbenatar) allows you to create new files from anywhere in your workspace. Press Ctrl + Alt + N, a list of paths to sub-directories will be listed for you to choose where you want to place your new file. Select the sub-directory, hit Enter, and type the name of the new file.

Markdown All In One

This extension (Yu Zhang) supports Markdown editing, rendering, previewing, and exporting.

Markdown Preview Enhanced

This extension by (Yiyi Wang) allows you to live preview the rendering of your Markdown file as you type. To open a preview tab (side by side), press Ctrl + K, V. One the preview tab, press Esc to toggle the table of contents.
