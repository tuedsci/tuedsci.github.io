What is VS Code?

Visual Studio Code (or VS Code) is a free, open-source, lightweight, and cross-platform code editor developed by Microsoft based on the Electron Framework. It is one of the most popular code editors used and recommended by programmers at all levels, from beginners to experienced professionals.

Features

Here are some of the highlighted features that have made VS Code popular.

  1. Free. You pay nothing to use it.
  2. Lightweight. Compared to comprehensive IDEs such as PyCharm or Eclipse, VS Code is much more lightweight. It launches very quickly and gives you a very smooth experience.
  3. Cross-platform. VS Code is available for all major operating systems, namely Windows, MacOS, and Linux.
  4. Multiple languages. VS Code supports a wide range of languages (e.g., Python, R, Java, C/C++) and file extensions (e.g., plain text, JSON, Markdown, HTML).
  5. IntelliSense. IntelliSense is an advanced feature offering smart code suggestions and completion based on the context of your code. It makes writing code faster, less prone to errors, and more fun.
  6. Extensions. VS Code has a massive extension repository, where you can find plugins for almost anything you want such as themes, code linters, or syntax highlighters. It makes VS Code extremely powerful and customizable.
  7. Command Palette. All functionality of VS Code can be accessed using the keyboard via the Command Palette. This not only allows you to code much faster but also reduces the amount of information you have to remember (because everything can be found very quickly by searching with the Command Palette)
  8. Other: Of course, as a mature code editor, VS Code provides all essential features such as syntax highlighting, code refactoring, integrated terminal, debugger, and version control.

Installation

To install VS Code, follow the steps below.

  1. Google the keywords download visual studio code, and you will find a link to download the installer. For now, you can find it here.
  2. Choose the appropriate installer for your operating system (e.g., Windows 11, 64-bit) and then download it.
  3. Click on the downloaded file and follow the instructions (mostly hit the Next button).
  4. After the installation process is completed, you can launch it and start writing code.
