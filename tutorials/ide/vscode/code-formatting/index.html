Code formatting

1 minutes to read
  • Installation
  • Configuration
  • Recommended settings

Code formatting refers to ways of organizing and structuring your code. It is an essential aspect of software development with the ultimate goal to make your codebase more consistent, readable, and maintainable. Prettier is an excellent extension that makes code formatting in VS Code much simpler. Follow the steps below to install and config Prettier.

Installation

Go to Extensions (Ctrl + Shift + X), search for Prettier - Code formatter, and install it.

Configuration

Set Prettier as your default formatter in settings.json (you can do this via GUI).

{
  "editor.defaultFormatter": "esbenp.prettier-vscode"
}

Recommended settings

To save time and typing, you can modify the settings to tell Prettier to automatically format your code whenever you save the file.

{
  "editor.formatOnSave": true
}
