Themes

You will spend a lot of time looking at the IDE while writing code. Thus, it is crucial to have a theme that is nice to the eyes. Fortunately, there are a lot of beautiful themes available on VS Code marketplace, and it is very easy to install and activate a new theme.

Types of themes

In VS Code, themes are categorized into 3 types as follows.

  1. Color themes. A color theme change both the GUI icons (e.g., icon for folders, file types, etc.) and the syntax highlighting.
  2. File Icon themes. A File Icon them only changes the GUI icons change.
  3. Product Icon themes. A Product Icon theme changes the icons on the main components of VS code such as the Activity Bar and the Status Bar.

Install new themes

Open Extensions (Ctrl + Shift + X) and search for the theme you want. After installing the theme, you can preview and decide if you want to activate it. See the next section.

Switching between themes

Switching between themes (any type) can be done via the Command Palette by searching for the keyword theme and then selecting the appropriate result from the dropdown list. For Color themes, it can be accessed quickly via the hotkey Ctrl + K Ctrl + T.

Recommended themes

Here are some of my favorite themes.

  • Color themes: One Dark Pro Flat, One Dark Pro, Atom One Dark, Gruvbox
  • Icon themes: Material Icon
  • Product Icon theme: Default, Fluent Icons
